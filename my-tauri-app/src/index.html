The #[tauri::command] macro bridges Rust functions to JavaScript. Type a name and call invoke("greet", { name }).

#[tauri::command]
fn greet(name: &str) -> String {
    format!("Hello, {}! Greeted from Rust!", name)
}

// JS: await invoke("greet", { name: "World" })

Shared state lives in Rust via Mutex<i32>. Commands read/write it using State<'_, AppState>. Persists for app lifetime.

Counter value lives in Rust memory, not JavaScript.

View Source 
struct AppState { counter: Mutex<i32> }

#[tauri::command]
fn update_counter(delta: i32, state: State<'_, AppState>) -> i32 {
    let mut c = state.counter.lock().unwrap();
    *c += delta;
    *c
}

Tauri passes Vec<Note> structs between Rust and JS seamlessly via serde serialization.

View Source 
#[derive(Serialize, Deserialize, Clone)]
struct Note { id: usize, text: String }

#[tauri::command]
fn add_note(text: String, state: State<'_, AppState>) -> Vec<Note> {
    let mut notes = state.notes.lock().unwrap();
    notes.push(Note { id, text });
    notes.clone()
}

Commands return Result<T, String>. Ok resolves the promise; Err rejects it. Catch errors with try/catch.

#[tauri::command]
fn divide(a: f64, b: f64) -> Result<f64, String> {
    if b == 0.0 {
        Err("Cannot divide by zero!".to_string())
    } else {
        Ok(a / b)
    }
}
